# Phase 11: LLM Engineering

Welcome to Phase 11. This phase transitions from training models to engineering systems around existing Large Language Models (LLMs). Here, you will learn how to build production-grade AI features like RAG, Vector Search, and Structured Data Extraction.

## Roadmap & Status

| Lesson | Status | Description |
|--------|--------|-------------|
| 01. Prompt Engineering | ✅ | System Prompts, Few-Shot, Chain of Thought |
| 02. Structured Outputs | ⬚ | Forcing JSON and Pydantic validation |
| 03. Embeddings & Vectors | ✅ | Translating text into mathematical representations |
| 04. Vector Databases | ⬚ | Storing and searching high-dimensional data (Qdrant, Pinecone) |
| 05. RAG Fundamentals | ✅ | Retrieval-Augmented Generation basics |
| 06. Advanced RAG | ⬚ | Re-ranking, HyDE, chunking strategies |

*Start with Lesson 01 and work your way down.*

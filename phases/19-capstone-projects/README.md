# Phase 19: Capstone Projects
End-to-end projects to prove your skills.

## Project Ideas
1. **Autonomous Coding Assistant:** An agent team that reads GitHub issues, writes code, tests it in a Docker container, and submits a PR.
2. **Enterprise RAG System:** A full-stack application with FastAPI, Qdrant, and streaming responses over company PDFs.
3. **Custom SLM (Small Language Model):** Fine-tune a 1B parameter model on a specific task (e.g., medical triage) using LoRA.
